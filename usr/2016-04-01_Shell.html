本文更新时间: `2016-04-01` - 本文的**目的**: 记录Shell脚本学习的一些特殊点，便于以后快速使用，免得学了又忘了 ### 赋值 - 赋值时**木有空格**: xx="hello" - 错误: xx = "hello" 提示: sh: xx: command not found - 引用: - echo $xx world - echo "$xx world" - echo "${xx} world" ### 执行命令并返回标准输出 - xx=$(ls -l); echo $xx; - xx=`ls -l`; echo $xx; ### 数学运算 - $[math calc] 可要空格，也可不要空格，**推荐这种写法** - a=1; b=$[$a+2]; echo $b; - ((math calc)) 可要空格，也可不要空格 - a=1; ((b=$a+2)); echo $b; - $((math calc)) 可要空格，也可不要空格 - a=1; b=$(($a+2)); echo $b; - expr命令后面都要空格 - a=1; b=$(expr $a + 2); echo $b; - a=1; b=$(expr $a "*" 2); echo $b; - let命令后面不要空格，如果加""，里面就可以用空格 - a=1; let b=$a+2; echo $b; - a=1; let "b = $a + 2"; echo $b; ### 比较 - ${varname} - $(get cmds std output) - $[math calc] - $((math calc)) ### 流程控制 ```shell if true ; then echo "hello" fi ``` ```shell if true ; then echo "hello" else echo "world" fi ``` ```shell if true ; then echo "hello" elif true ; then echo "world" fi ``` ### 测试 - test 如果表达式为true，退出状态为0，否则为1 - test expression - [ expression ] **注意**这里需要空格 - test 命令有很多选项，可以判断文件，字符串，整数等，具体搜索 shell test命令 - if [ 2 -gt 3 ] ; then echo "true" ; else echo "false"; fi; - [[ expression ]] - exit 0 表示正常，额，和C一样 ### case - case 匹配字符串，不是数字 ```shell case $argA in hello) echo "hello" ;; world) echo "world" ;; *) break ;; esac ``` ### 循环 ```shell while true ; do echo hello done ``` ```shell for var in $List ; do ... done ``` - 未测试: List 是个用空格分隔的列表 **参数**: - $0 脚本名 - $1 第1个输入的参数(函数，或命令行) - $2 第2个输入的参数 - $# 参数个数 - $@ - 主程序下是以空格分隔的所有命令行参数 - 函数中是逗号分隔的所有函数参数
